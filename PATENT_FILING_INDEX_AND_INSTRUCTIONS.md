## How to File These Patents

### Step 1: Choose Filing Platform

**Option A - USPTO (United States Patent and Trademark Office)**
- Website: https://www.uspto.gov/patents/patent-process
- File: Provisional Patent Application (PPA)
- Cost: $320 (small entities)
- Time to file: ~1 hour per patent

**Option B - WIPO (World Intellectual Property Organization)**
- Website: https://www.wipo.int/
- File: International Provisional Patent Application
- Cost: Varies by country (~$2,000-5,000 per patent)
- Coverage: ~200 countries

**Option C - Patent Filing Service**
- LegalZoom, Rocket Lawyer, or local patent attorney
- Cost: $300-1,000 per patent + professional fees
- Benefit: Professional review and guidance

---

### Step 2: Prepare Each Patent for Filing

For **USPTO Filing (Easiest DIY Option)**:

1. **Convert to PDF Format**
   - Open each .md file
   - Copy content to Microsoft Word or Google Docs
   - Export as PDF
   - Ensure formatting is clean

2. **Create Cover Sheet (Form SB/16)**
   - Download from: https://www.uspto.gov/patents/patent-forms
   - Fill in:
     - Applicant name (you)
     - Address
     - Patent title (exact from document)
     - Inventors (list all if joint inventors)
     - Attorney/agent info (if using)

3. **Prepare Filing Package for EACH Patent:**
   - Cover sheet (Form SB/16)
   - Patent document (PDF)
   - Any drawings/figures (if referenced - optional for PPA)
   - Fee payment

4. **Submit Online via USPTO EFS-Web**
   - Go to: https://www.uspto.gov/patents/apply/e-filing
   - Upload all files
   - Pay filing fee ($320 per patent for small entities)
   - Receive confirmation with application number
