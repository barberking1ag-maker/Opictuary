## ‚úÖ STEP 6: Upload App Bundle

‚ö†Ô∏è **IMPORTANT**: You need to build the Android App Bundle (.aab file) first.

**Quick Build Instructions**:

1. **Install Android Studio** on your computer (not on Replit)
2. **Download your Replit project** to your computer
3. **Open Terminal/Command Prompt** and run:

```bash
# Build web app
npm run build

# Sync to Android
npx cap sync android

# Navigate to android folder
cd android

# Build release bundle
./gradlew bundleRelease
```

4. **Find your AAB file** at:
   `android/app/build/outputs/bundle/release/app-release.aab`

**Upload to Play Console**:

1. Navigate to **Release** ‚Üí **Production**
2. Click **Create new release**
3. Under **App integrity**, select:
   ```
   Use Google Play App Signing (recommended)
   ```
4. Click **Upload** and select `app-release.aab`
5. Wait for processing (may take 2-5 minutes)
