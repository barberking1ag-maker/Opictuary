**For Cover Sheet (Each Patent):**
- Your full legal name
- Your complete mailing address

- Your phone number
- Citizenship: United States

**Title of Invention (Copy from each patent file):**
- Patent 1: "SECURE PRISON ACCESS SYSTEM FOR DIGITAL MEMORIAL PLATFORMS WITH IDENTITY VERIFICATION AND TIME-LIMITED ACCESS TOKENS"

- Patent 5: "CATEGORY-SPECIFIC MEMORIAL CREATION SYSTEM FOR ESSENTIAL WORKERS AND FIRST RESPONDERS WITH PROFESSION-TAILORED FIELDS"
- Patent 6: "MULTI-CHANNEL REVENUE SYSTEM FOR DIGITAL MEMORIAL PLATFORMS WITH INTEGRATED PARTNERSHIPS, PRISON SERVICES, AND TIERED COMMISSION STRUCTURE"
- Patent 7: "COMPREHENSIVE CELEBRITY MEMORIAL VERIFICATION AND ESTATE MANAGEMENT SYSTEM WITH MULTI-STEP APPROVAL AND EXCLUSIVE FAN CONTENT"

### STEP 4: Fill Out USPTO Form PTO/SB/16
**Form Link:** https://www.uspto.gov/sites/default/files/documents/sb0016.pdf

**For Each Patent, Fill In:**
- Docket Number: Use the one in each patent file (OPI-001-PRISON, OPI-002-QR, etc.)
- Title: Copy from patent file
- Inventor(s): Your name and address
- Correspondence Address: Your address
- Application Type: Provisional Patent Application

### STEP 5: Prepare Written Description
**For each patent:**
- Copy entire content from patent .md file
- Paste into Microsoft Word or Google Docs
- Convert to PDF
- Name file: "OPI-001-PRISON-Description.pdf" (use patent number)

**Total PDFs to create: 7** (one per patent)

### STEP 6: File Online via USPTO
**Electronic Filing System (EFS-Web):**
https://sportal.uspto.gov/#/

**Upload for Each Patent:**
1. Cover sheet (Form PTO/SB/16) - PDF
2. Written description - PDF (from Step 5)
3. Select entity status: **Micro Entity**
4. Pay filing fee: **$65**

**Payment Methods:**
- Credit/debit card
- USPTO deposit account
- Electronic Funds Transfer

### STEP 7: Receive Confirmation
**Immediately after filing:**
- Email confirmation with application number
- Application number format: 63/XXX,XXX
- Receipt showing filing date
- "Patent Pending" status begins IMMEDIATELY

**Save these receipts!** Proof of filing date.
