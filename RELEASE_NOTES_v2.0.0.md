   ```bash
   git add -A

   git push origin main
   ```

2. **GitHub Actions will automatically:**
   - Build the production frontend
   - Sync with Capacitor
   - Generate signed AAB file
   - Upload as artifact

3. **Download AAB:**
   - Go to GitHub repository â†’ Actions tab
   - Find the latest workflow run
   - Download `app-release-bundle` artifact
   - Extract to get `app-release.aab`

### Uploading to Google Play Console

1. **Login to Play Console:**

   - Select Opictuary app
