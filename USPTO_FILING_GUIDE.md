## WHAT HAPPENS AFTER FILING

### Immediate (Within 1 Hour)
- ‚úÖ Email confirmation with application numbers
- ‚úÖ Filing receipts available for download
- ‚úÖ **YOU CAN NOW USE "PATENT PENDING"**

### Within 1-4 Weeks
- USPTO assigns examiner (for tracking purposes only)
- Application enters USPTO database
- You can search for it on USPTO.gov Public PAIR

### Nothing Else for 12 Months
- Provisional patents are **NOT examined**
- No office actions, no rejections, no correspondence
- USPTO holds your filing date for 12 months
- You simply have "Patent Pending" status during this time

### At 12 Months - Decision Point

**Option A: File Utility Patents ($10K-$25K each with attorney)**
- If you've raised funding
- Convert provisionals to full utility patents
- Maintain original filing date (priority date)
- USPTO examines and potentially grants full patents

**Option B: Let Provisionals Expire**
- If you haven't raised funding or pivoted
- Provisionals automatically expire at 12 months
- No refund of $150
- Innovations enter public domain
- Can no longer use "Patent Pending"

**Option C: File Second Provisionals ($150 again)**
- Rare approach: file new provisionals for another 12 months
- Loses original priority date
- Useful if you've made major improvements

## FAQ

**Q: Can I file these myself without a lawyer?**  
A: Yes! Provisional patents are designed for DIY filing. You've already written everything needed. Full utility patents later will need a lawyer.

**Q: What if I make a mistake in the application?**  
A: Provisional patents aren't examined, so minor errors don't matter. You can correct everything when/if you file utility patents.

**Q: Can I add features to the platform that aren't in the patents?**  
A: Yes, but those features won't have "Patent Pending" protection. File new provisionals for major new innovations.

**Q: Do I need to draw the diagrams?**  
A: No. The text descriptions in PATENT_*_DRAWINGS.md are sufficient for provisional filing. Full utility patents need actual drawings.

**Q: When do patents actually get granted?**  
A: Not from provisionals. Only if you file utility patents later (12-36 months after filing utility, so 2-4 years total).

**Q: Can I sell the platform with "Patent Pending"?**  
A: Yes. Patent Pending is attractive to acquirers as it shows IP protection and option to complete full patents.

**Q: What if someone copies my technology?**  
A: If you later get full patents granted, the patent protection dates back to your provisional filing date. You could potentially sue for infringement.

**Q: Can I still improve the inventions after filing?**  
A: Yes! Keep building. File continuation provisionals or add to utility patents later.
