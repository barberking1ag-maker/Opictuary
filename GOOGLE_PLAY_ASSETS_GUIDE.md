### How to Capture Screenshots (Step-by-Step)

#### Method 1: Chrome DevTools (Recommended)

1. **Open Opictuary in Chrome**
   ```bash
   # Your app should be running on http://localhost:5000
   ```

2. **Open DevTools**
   - Press `F12` or `Cmd+Option+I` (Mac) / `Ctrl+Shift+I` (Windows)
   - Click the device toggle icon (or press `Cmd+Shift+M`)

3. **Set Device Dimensions**
   - Select device: "iPhone 12 Pro" (1170 x 2532)
   - Or custom: 1080 x 1920 pixels

4. **Navigate to Page**
   - Go to the page you want to screenshot
   - Ensure page is fully loaded and looks good

5. **Capture Screenshot**
   - Open Command Menu: `Cmd+Shift+P` (Mac) / `Ctrl+Shift+P` (Windows)
   - Type "Capture screenshot"
   - Select "Capture screenshot" (captures visible area)
   - Or "Capture full size screenshot" (captures entire page)

6. **Save File**
   - Screenshot downloads as PNG
   - Rename to descriptive name (e.g., `screenshot_1_home.png`)

---

#### Method 2: Real Android Device (Alternative)

1. **Install Opictuary on Android device**
   - Build APK (see Android Build Guide)
   - Install on your phone

2. **Navigate to screens**
   - Open each page you want to screenshot

3. **Capture Screenshot**
   - Android: Press `Power + Volume Down` simultaneously
   - Screenshots save to Photos app

4. **Transfer to Computer**
   - Use USB cable or cloud sync (Google Photos, Dropbox)
   - Download screenshots to computer
