# PROVISIONAL PATENT APPLICATION

## Patent Title: Privacy-Preserving QR Code Analytics System for Memorial Engagement

---

## ABSTRACT

A computer-implemented system for tracking and analyzing QR code engagement metrics while maintaining user privacy through data anonymization, location limiting, and consent-based tracking. The system generates meaningful insights about memorial popularity and engagement without compromising individual user privacy.

---

## BACKGROUND OF THE INVENTION

This invention relates to analytics and privacy protection. Prior art analytics systems compromise user privacy. This invention provides analytics without privacy violation.

---

## SUMMARY OF THE INVENTION

System enabling:
1. QR scan tracking without user identification
2. Location data limited to city level
3. Device type recording without device ID
4. Consent-based analytics opt-in
5. Aggregate reporting only
6. No cross-product user tracking
7. Automatic data retention limits
8. Privacy compliance (GDPR, CCPA)

---

## DETAILED DESCRIPTION

### Privacy Mechanisms
- No permanent user identifiers stored
- Location limited to city, not GPS coordinates
- Device type anonymized (not IMEI or MAC address)
- Session-based tracking only
- No cookies or persistent trackers
- Automatic data deletion after retention period
- Opt-in consent before tracking

### Analytics Provided
- Total scans by time period
- Geographic scan distribution (city-level)
- Device type distribution
- Time-of-day patterns
- Repeat scan detection (anonymous)
- Engagement trends

### Compliance
- GDPR compliant
- CCPA compliant
- State privacy law compliant
- Privacy by design
- Data minimization
- User consent required

## CLAIMS

**Claim 1:** A privacy-preserving analytics system for QR codes comprising:
- scan tracking without persistent user identification;
- location data limited to city-level only;
- device type recording without device identifiers;
- session-based tracking with automatic expiration;
- consent mechanism requiring user opt-in;
- no cross-product user tracking;
- automatic data retention limits;
- aggregate reporting only.

**Claim 2:** The system of Claim 1, wherein privacy is preserved by:
- generating random session IDs;
- storing city-level location only;
- anonymizing device types;
- not storing IP addresses;
- not using cookies or persistent identifiers;
- implementing automatic data deletion.

**Claim 3:** A method for analytics collection comprising:
- obtaining user consent before tracking;
- recording scan timestamp only;
- recording anonymized device type;
- recording city-level location;
- generating aggregate statistics;
- preventing user re-identification;
- automatically deleting data after retention period.
