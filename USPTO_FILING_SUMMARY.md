## ðŸ“‹ REQUIRED USPTO FORMS

### For Each Provisional Patent Application

#### 1. **Form PTO/SB/16** - Provisional Application Cover Sheet
- Download: [USPTO Form SB/16](https://www.uspto.gov/sites/default/files/documents/sb0016.pdf)
- Purpose: Cover sheet for provisional application
- Required Information:
  - Title of invention
  - Inventors' names and addresses
  - Correspondence address
  - Entity status (micro/small/large)

#### 2. **Application Data Sheet (ADS) - Form PTO/AIA/14**
- Download: [USPTO Form AIA/14](https://www.uspto.gov/sites/default/files/documents/aia0014.pdf)
- Purpose: Bibliographic data
- Required Information:
  - Applicant information
  - Inventor information
  - Correspondence information
  - Representative information (if applicable)

#### 3. **Entity Status Form**
- **Micro Entity: Form PTO/SB/15A**
  - Certifies micro entity qualification
  - Must meet income and application limits
- **Small Entity: Form PTO/SB/15B**
  - For businesses with <500 employees

#### Optional Forms
- **Power of Attorney (PTO/AIA/82)** - If using patent attorney
- **Nonpublication Request (PTO/SB/35)** - To prevent publication

## ðŸš€ STEP-BY-STEP FILING PROCESS

### Step 1: Create USPTO Account
```
1. Go to https://www.uspto.gov
2. Click "Patents" â†’ "File Online"
3. Register for USPTO.gov account
4. Verify email address
5. Set up two-factor authentication
```

### Step 2: Prepare Documents for Each Patent
```
For each patent, prepare:
1. Specification document (from USPTO_PATENT_FILINGS.md)
2. Cover sheet (Form SB/16)
3. Application Data Sheet (Form AIA/14)
4. Entity status certification (Form SB/15A)
```

### Step 3: File via EFS-Web
```
1. Log into Patent Center
2. Select "File a new application"
3. Choose "Provisional"
4. Upload documents:
   - Specification (PDF)
   - Cover sheet (PDF)
   - ADS (PDF)
   - Entity certification (PDF)
5. Review filing
6. Pay fees ($65 per patent)
7. Submit application
8. Save confirmation number
```

### Step 4: Post-Filing Actions
```
Immediately after filing:
- Download filing receipt
- Note application number
- Update website with "Patent Pending"
- Update marketing materials
- Inform investors

Within 2 weeks:
- Receive official filing receipt
- Review for accuracy
- File any corrections if needed
```
